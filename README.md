Workout Log არის Android აპლიკაცია, რომელიც განკუთვნილია სავარჯიშო დარბაზში შესრულებული ვარჯიშების აღრიცხვისთვის. მისი საშუალებით შეგიძლიათ ჩაიწეროთ სავარჯიშოები, ნახოთ ვარჯიშების ისტორია, ავტომატურად დააფიქსიროთ პირადი რეკორდები და შეინახოთ ვარჯიშის შაბლონები სწრაფად განმეორებითი გამოყენებისთვის. ყველა მონაცემი ინახება მხოლოდ მოწყობილობაზე და არ საჭიროებს ანგარიშს ან ინტერნეტთან კავშირს.

რას აკეთებს აპლიკაცია

აპლიკაციას აქვს 5 ეკრანი, რომლებზეც გადასვლა შესაძლებელია ქვედა ნავიგაციის ზოლის მეშვეობით.

Today ეკრანი აჩვენებს მიმდინარე დღისთვის ჩაწერილ ყველა სავარჯიშოს. ზედა ნაწილში ნაჩვენებია შეჯამება: რამდენი სავარჯიშო შეასრულეთ, რამდენი სეტი გააკეთეთ ჯამში და რა იყო საერთო სავარჯიშო მოცულობა (Training Volume). ქვემოთ განთავსებულია დღევანდელი ყველა სავარჯიშოს გადახვევადი სია. ნებისმიერი ჩანაწერის წაშლა შესაძლებელია ნაგვის ურნის ხატულაზე დაჭერით.

Log ეკრანი წარმოადგენს ფორმას, რომლის საშუალებითაც ამატებთ ახალ სავარჯიშოს. საჭიროა შეიყვანოთ სავარჯიშოს დასახელება, სეტების რაოდენობა, გამეორებების რაოდენობა (Reps), წონა კილოგრამებში და სურვილის შემთხვევაში შენიშვნები. შენახვამდე ფორმა ამოწმებს ყველა ველის სისწორეს და თუ რომელიმე ველი ცარიელია ან არასწორად არის შევსებული, შესაბამის შეცდომის შეტყობინებას აჩვენებს. წარმატებით შენახვის შემდეგ გამოჩნდება Snackbar შეტყობინება, რომელიც ადასტურებს შენახვას. თუ შეყვანილი წონა აღემატება ამ სავარჯიშოზე თქვენს წინა საუკეთესო შედეგს, გამოჩნდება მეორე Snackbar, რომელიც გაცნობებთ ახალი პირადი რეკორდის დამყარების შესახებ.

History ეკრანი აჩვენებს თქვენს მიერ ოდესმე ჩაწერილ ყველა ვარჯიშს. ჩანაწერები დაჯგუფებულია თარიღების მიხედვით, სადაც ყველაზე ახალი ჩანაწერები სიის თავშია. თითოეული განყოფილება მონიშნულია როგორც Today, Yesterday ან ძველი ჩანაწერებისთვის სრული თარიღით.

PRs ეკრანი აჩვენებს თითოეული სავარჯიშოს პირად რეკორდს. რეკორდად ითვლება ყველაზე დიდი წონა, რომელიც ოდესმე დაგიფიქსირებიათ კონკრეტულ სავარჯიშოზე. ეს მონაცემები ავტომატურად განახლდება ყოველ ჯერზე, როდესაც ახალ ვარჯიშს შეინახავთ. თითოეული ჩანაწერი შეიცავს სავარჯიშოს დასახელებას, წონას, ამ წონაზე შესრულებული გამეორებების რაოდენობას და რეკორდის დაფიქსირების თარიღს.

Templates ეკრანი საშუალებას გაძლევთ შეინახოთ სავარჯიშოების წინასწარ გამზადებული შაბლონები, სადაც მითითებულია ნაგულისხმევი სეტების რაოდენობა, გამეორებები და წონა. შაბლონის Play ღილაკზე დაჭერისას Log ფორმა ავტომატურად ივსება შესაბამისი მონაცემებით და აპლიკაცია გადაგიყვანთ ამ ეკრანზე, რაც მნიშვნელოვნად ამცირებს ერთი და იმავე სავარჯიშოების ხელახლა შეყვანისთვის საჭირო დროს.
